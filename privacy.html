Privacy Policy

Last updated: August 8, 2025

1. About this website

This website (chapter322.github.io/FreeTapWater) is a personal, non-commercial project operated by Chapter322 (GitHub username) to collect and display publicly shared information about restaurants and cafes that offer free or reasonably priced tap water.

2. Purpose

The purpose of this website is to inform visitors and allow them to share relevant public information about hospitality venues’ tap water policies. This is done purely for informational and non-commercial purposes.

3. Personal data

This website does not intentionally collect personal data from visitors. However:

  • The displayed restaurant data (name, location, address) may, in some cases, identify a natural person — for example, in the case of sole proprietorships or personally owned cafes.
  • This data is publicly available and generally already shared by the establishments themselves, but under the GDPR it can still be considered personal data.
  • In such cases, the website acts as both the data controller and processor of the published information.

4. Data sources

The information about restaurants is collected from public submissions, online resources, and publicly available information (e.g., Google Maps, OpenStreetMap).

5. Third-party services

This website uses:

  • GitHub Pages for hosting. GitHub may log IP addresses and request data for security and analytics.
  • OpenStreetMap embedded via an iframe to display locations. OpenStreetMap servers may log IP addresses.
  • Google Fonts for typography. Fonts are served by Google, which may log requests and associated IPs.

6. Your rights

If you are a data subject under GDPR and believe that the website is displaying personal data about you, you can request correction or removal by contacting Chapter322 via GitHub.

7. Disclaimer

This is a personal project with no commercial intent. While the information is collected and maintained with care, no guarantee is given as to its accuracy or completeness.
